Commandes

Liste complète des commandes disponibles

Utilise /help en jeu pour voir toutes les commandes disponibles.

Commandes générales

Commande Description
/ob create <nom> Pour commencer ton aventure OneBlock
/ob panel Ouvre le panel OneBlock pour en savoir plus
/rang Pour débloquer des nouveaux grades
/boutique Pour dépenser tes Crédits
/wiki Si tu as besoin d'aide
/spawn Téléportation au spawn principal
/vote Affiche les liens de vote
/kit Ouvre le menu des kits disponibles
/daily Récupère ta récompense quotidienne
/link Lie ton compte Minecraft à Discord

Commandes OneBlock

Commande Description
/ob Téléportation sur ton île
/ob create Crée une nouvelle île
/ob upgrade Améliore le niveau de ton île
/ob invite <joueur> Invite un joueur sur ton île
/ob kick <joueur> Expulse un joueur de ton île
/ob top Affiche le classement des îles
/ob settings Ouvre les paramètres de ton île
/ob members Affiche la liste des membres de ton île
/ob visit <joueur> Visite l'île d'un autre joueur
/ob lock Verrouille ton île (empêche les visites)
/ob biome <biome> Change le biome de ton île
/ob border Affiche les bordures de ton île

Commandes de téléportation

Commande Description
/tpa <joueur> Envoie une demande de téléportation à un joueur
/tpaccept Accepte une demande de téléportation
/tpdeny Refuse une demande de téléportation
/home Téléportation à ton home
/sethome <nom> Définit un point de home
/delhome <nom> Supprime un point de home
/homes Liste tous tes homes
/back Retourne à ta dernière position
/warp <nom> Téléportation à un warp public

Commandes économie

Commande Description
/bal Affiche ton solde actuel
/pay <joueur> <montant> Envoie de l'argent à un joueur
/baltop Classement des joueurs les plus riches
/shop Ouvre le menu du shop
/ah Ouvre l'hôtel des ventes
/ah sell <prix> Met en vente l'objet dans ta main
/autosell Active/désactive la vente automatique
/trade <joueur> Propose un échange sécurisé avec un joueur

Commandes sociales

Commande Description
/msg <joueur> <message> Envoie un message privé à un joueur
/r <message> Répond au dernier message privé reçu
/party create <nom> Crée un groupe de joueurs
/party invite <joueur> Invite un joueur dans ton groupe
/party leave Quitte le groupe actuel
/party chat Bascule vers le chat de groupe
/ignore <joueur> Ignore les messages d'un joueur
/mail send <joueur> <msg> Envoie un mail hors-ligne à un joueur
/mail read Lit tes mails reçus
